java中的静态代码块

本文详细说明 java 静态代码块。

静态代码块

执行优先级高于非静态的初始化块。

它会在类初始化的时候执行一次且只执行一次，执行完成便销毁。

它仅能初始化类变量，即 static 修饰的数据成员，使得不需要每次调用这个变量时都给它赋值。

非静态代码块

执行的时候如果有静态初始化块，先执行静态初始化块再执行非静态初始化块。

在每个对象生成时都会被执行一次，它可以初始化类的实例变量。

非静态初始化块会在构造函数执行时，在构造函数主体代码执行之前被运行。
